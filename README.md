# 😀 Facial Expression Recognition using CNN

This project implements a Facial Expression Recognition system using a Convolutional Neural Network (CNN).  
The model analyzes facial images and predicts the emotion displayed by the person.

Facial expression recognition is a computer vision task where deep learning models classify human emotions from facial features. :contentReference[oaicite:0]{index=0}  

---

## 🚀 Features

- Detects facial expressions from images  
- Uses CNN for emotion classification  
- Preprocessing of facial images  
- Predicts human emotions automatically  

---

## 🧠 Technologies Used

- Python  
- Deep Learning (CNN)  
- TensorFlow / Keras  
- OpenCV  
- NumPy & Pandas  

---

## 😊 Recognized Emotions

The model can classify common facial emotions such as:

- Angry  
- Disgust  
- Fear  
- Happy  
- Sad  
- Surprise  
- Neutral  

---

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/kowshik102007/Facial-Expression-Recognition-using-CNN.git
cd Facial-Expression-Recognition-using-CNN
```

2. Install required libraries:

```
pip install -r requirements.txt
```

3. Run the prediction or main script:

```
python main.py
```

*(Use the correct script name from your project files if different.)*
